# ADR-0001: Project Scope

## Status

Accepted

## Context

A generic “chat with any document” system is too broad for rigorous research and evaluation.

## Decision

PaperLens will focus on scientific and technical papers, with explicit support for text, tables, figures, charts, equations, and page layout.

## Consequences

- The system can use domain-specific evaluation questions.
- The architecture remains general enough to support other document domains later.
- Product claims are narrower but more testable.
