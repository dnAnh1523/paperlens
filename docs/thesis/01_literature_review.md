# Literature Review

The literature review should cover the areas PaperLens depends on and the areas it intentionally has
not implemented yet.

Current review sections to develop:

- Retrieval-augmented generation for document question answering.
- Scientific document parsing and text-layer PDF extraction.
- Lexical retrieval baselines, including BM25 and SQLite FTS5.
- Citation grounding, source inspection, and evidence provenance.
- Evaluation metrics for retrieval, including `hit@k`, MRR, and no-result query counts.
- Multimodal document AI for tables, figures, page images, captions, equations, and layout.
- OCR and scanned-PDF processing tradeoffs.
- Answer faithfulness and citation-constrained generation.

Current M1-M18 implementation evidence should be cited only for local retrieval foundations and
evidence inspectability. It should not be cited as completed multimodal QA, OCR, table extraction,
figure understanding, or LLM answer synthesis.
