# Experiments

The current experiment record is maintained in:

- `docs/thesis/EXPERIMENT_LOG.md`

Implemented experiment categories:

- Retrieval smoke test for fixture seeding, ingestion, chunking, indexing, and retrieval plumbing.
- Retrieval benchmark v1 for natural-language local lexical retrieval with distractors.
- LIKE vs FTS5 vs AUTO comparison.
- JSON and Markdown report generation.

The smoke test is not a benchmark. Benchmark v1 is synthetic and should be treated as early local
failure analysis, not final scientific-paper QA evidence.
